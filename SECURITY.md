# Security Policy

If you find a security issue with the `CLAUDE.md` instructions or the `altman-grader` skill — for example a prompt that causes unsafe behavior, a way to leak sensitive context, or a way to make Claude execute something it shouldn't — please report it **privately**.

**Open a security advisory:**
https://github.com/Tarcroi/altman-mode/security/advisories/new

Please do not open public issues for security problems.
